Off-policy Methods with Approximation

Posted Aug 30, 2022 Updated Aug 31, 2022
By DevSlem
11 min read

мқҙ нҸ¬мҠӨнҠём—җм„ңлҠ” on-policy function approximationмқ„ off-policyлЎңмқҳ нҷ•мһҘкіј мқҙлЎң мқён•ҙ л°ңмғқн•ҳлҠ” л¬ём ңл“Өм—җ лҢҖн•ҙ лӢӨлЈ° кІғмқҙлӢӨ.

Introduction

off-policy methodлҠ” behavior policy $b$м—җ мқҳн•ҙ мғқм„ұлҗң experienceлЎңл¶Җн„° target policy $\pi$м—җ лҢҖн•ң value functionмқ„ н•ҷмҠөн•ҳлҠ” л°©лІ•мқҙлӢӨ. л°ҳлҢҖлЎң on-policy methodлҠ” behavior policyмҷҖ target policyк°Җ лҸҷмқјн•ң л°©лІ•мқҙлӢӨ. off-policy methodм—җм„ңлҠ” мқјл°ҳм ҒмңјлЎң target policy $\pi$лҠ” greedy policyмқҙкі , behavior policy $b$лҠ” мўҖ лҚ” нҡЁкіјм ҒмңјлЎң explorationмқ„ мҲҳн–үн•  мҲҳ мһҲлҠ” policy (e.g., $\epsilon$-greedy policy) мқҙлӢӨ.

off-policy methodмқҳ л¬ём ңлҠ” нҒ¬кІҢ л‘җк°Җм§ҖлЎң лӮҳлүңлӢӨ. мІ« лІҲм§ёлҠ” update target (target policy м•„лӢҳ)кіј кҙҖл Ёлҗң мқҙмҠҲлЎң, мқҙм „ tabular off-policy methodм—җм„ң лӢӨлӨҳм—ҲлҚҳ importance samplingкіј кҙҖл Ёлҗҳм–ҙ мһҲлӢӨ. л‘җ лІҲм§ёлҠ” update distributionкіј кҙҖл Ёлҗң мқҙмҠҲлЎң, update distributionмқҙ on-policy distributionмқ„ л”°лҘҙм§Җ м•ҠлҠ” лӢӨлҠ” м җмқҙлӢӨ. on-policy distributionмқҖ semi-gradient methodмқҳ м•Ҳм •м„ұм—җ мһҲм–ҙ мӨ‘мҡ”н•ҳлӢӨ. мқҙ л‘җ мқҙмҠҲм—җ лҢҖн•ҙ мқҙлІҲ нҸ¬мҠӨнҠём—җм„ң лӢӨлЈ° кІғмқҙлӢӨ.

Semi-gradient Methods

мқҙ м„№м…ҳм—җм„ңлҠ” off-policy methodлҘј к°„лһөнһҲ semi-gradient methodлЎң нҷ•мһҘн•ңлӢӨ. мқҙ нҷ•мһҘлҗң л°©лІ•мқҖ off-policy methodмқҳ мІ« лІҲм§ё мқҙмҠҲм—җ лҢҖн•ҙм„ңл§Ң лӢӨлЈЁл©°, мҲҳл ҙмқҙ м•Ҳлҗҳм–ҙ л°ңмӮ°н•  л•ҢлҸ„ мһҲлӢӨ.

$n$-step tabular methodм—җм„ң мӮ¬мҡ©н–ҲлҚҳ л°©лІ•мқ„ лӢЁмҲңнһҲ function approximationмқ„ мӮ¬мҡ©н•ң weight vector $\mathbf{w}$м—җ лҢҖн•ң updateлЎң ліҖкІҪн•ңлӢӨ. off-policy methodлҠ” target policyмҷҖ behavior policyмқҳ distributionмқҙ лӢӨлҘҙкё° л•Ңл¬ём—җ importance sampling кё°лІ•мқ„ мӮ¬мҡ©н•ңлӢӨ.1 к·ё мӨ‘ off-policy TD(0)мҷҖ к°ҷмқҖ м•Ңкі лҰ¬мҰҳмқҖ per-step importance sampling ratioлҘј мӮ¬мҡ©н•ңлӢӨ.

\[\rho_t \doteq \rho_{t:t} = \dfrac{\pi(A_t \vert S_t)}{b(A_t \vert S_t)}\]

м•„лһҳлҠ” off-policy semi-gradient TD(0)лЎң semi-gradient on-policy TD(0)м—җ $\rho_t$л§Ң м¶”к°Җлҗҳм—ҲлӢӨ.

\[\mathbf{w}_{t+1} \doteq \mathbf{w}_t + \alpha \rho_t \delta_t \nabla \hat{v}(S_t, \mathbf{w}_t)\]

$\delta_t$лҠ” TD errorлЎң м•„лһҳлҠ” к°Ғк°Ғ episodic, discounted settingкіј average rewardлҘј мӮ¬мҡ©н•ҳлҠ” continuing, undiscounted settingм—җм„ңмқҳ TD errorмқҙлӢӨ.

\[\begin{align*} & \delta_t \doteq R_{t+1} + \gamma \hat{v}(S_{t+1}, \mathbf{w}_t) - \hat{v}(S_t, \mathbf{w}_t) \text{, or} \tag{episodic} \\ & \delta_t \doteq R_{t+1} + \bar{R}_t + \hat{v}(S_{t+1}, \mathbf{w}_t) - \hat{v}(S_t, \mathbf{w}_t) \tag{continuing} \end{align*}\]

$\bar{R}_t$лҠ” average rewardмқҳ м¶”м •м№ҳмқҙлӢӨ.2

action valueм—җ лҢҖн•ҙм„ңлҸ„ мүҪкІҢ м „нҷҳн•  мҲҳ мһҲлӢӨ. м•„лһҳлҠ” off-policy semi-gradient Expected SarsaмқҙлӢӨ.

\[\begin{align*} & \mathbf{w}_{t+1} \doteq \mathbf{w}_t + \alpha \delta_t \nabla \hat{q}(S_t, A_t, \mathbf{w_t}) \text{, with} \\ \\ & \delta_t \doteq R_{t+1} + \gamma \sum_a \pi(a \vert S_{t+1}) \hat{q}(S_{t+1}, a, \mathbf{w}_t) - \hat{q}(S_t, A_t, \mathbf{w}_t) \text{, or} \tag{episodic} \\ & \delta_t \doteq R_{t+1} - \bar{R}_t + \sum_a \pi(a \vert S_{t+1}) \hat{q}(S_{t+1}, a, \mathbf{w}_t) - \hat{q}(S_t, A_t, \mathbf{w}_t) \tag{continuing} \end{align*}\]

мқҙ м•Ңкі лҰ¬мҰҳмқҖ importance samplingмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ”лӢӨ. tabular caseм—җм„ңлҠ” action valueм—җ лҢҖн•ң 1-step TD methodм—җ importance samplingмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” мқҙмң к°Җ лӘ…нҷ•н–ҲлӢӨ.3 к·ёлҹ¬лӮҳ function approximationм—җм„ңлҠ” лӘ…нҷ•н•ҳм§Җ м•ҠлӢӨ. к·ё мқҙмң лҘј Sutton мұ…м—җм„ңлҠ” м•„лһҳмҷҖ к°ҷмқҙ м„ӨлӘ…н•ңлӢӨ. (мҶ”м§ҒнһҲ лӮҙк°Җ мқҙн•ҙ лӘ»н–ҲмқҢ)

With function approximation it is less clear because we might want to weight different state-action pairs differently once they all contribute to the same overall approximation.4

м•„лһҳлҠ” off-policy semi-gradient Sarsaмқҳ $n$-step versionмқҙлӢӨ.

\[\begin{align*} & \mathbf{w}_{t+n} \doteq \mathbf{w}_{t+n-1} + \alpha \rho_{t+1} \cdots \rho_{t+n} [G_{t:t+n} - \hat{q}(S_t, A_t, \mathbf{w}_{t+n-1})] \nabla \hat{q}(S_t,A_t,\mathbf{w}_{t+n-1}) \text{, with} \\ \\ & G_{t:t+n} \doteq R_{t+1} + \cdots + \gamma^{n-1}R_{t+n} + \gamma^n \hat{q}(S_{t+n}, A_{t+n}, \mathbf{w}_{t+n-1}) \text{, or} \tag{episodic} \\ & G_{t:t+n} \doteq R_{t+1} - \bar{R}_t + \cdots + R_{t+n} - \bar{R}_{t+n-1} + \hat{q}(S_{t+n}, A_{t+n}, \mathbf{w}_{t+n-1}) \tag{continuing} \end{align*}\]

мң„ мІ« лІҲм§ё мҲҳмӢқм—җм„ң $\rho_k$лҠ” $k \geq T$ ($T$лҠ” episodeмқҳ terminal step) мқј л•Ң $1$мқҙлӢӨ. $t + n \geq T$мқј л•Ң $G_{t:t+n} \doteq G_t$мқҙлӢӨ.

м„ё лІҲм§ё мҲҳмӢқм—җм„ң on-policy semi-gradient $n$-step Sarsaмқҳ кІҪмҡ° average rewardк°Җ мҳӨм§Ғ $\bar{R}_{t+n-1}$мқҙм—ҲлҠ”лҚ° м—¬кё°м„ңлҠ” лӢӨлҘҙлӢӨ. off-policy м—¬м„ң лӢӨлҘё кұҙм§Җ, мұ…мқҙ мҳӨлҘҳмқё кұҙм§Җ лӘЁлҘҙкІ лӢӨ.

The Deadly Triad

off-policy learningм—җм„ң л°ңмғқн•ҳлҠ” л¶Ҳм•Ҳм •м„ұкіј л°ңмӮ° мқҙмҠҲлҠ” м•„лһҳ 3к°Җм§Җ мҡ”мҶҢлҘј лӘЁл‘җ кІ°н•©н•  л•Ң л°ңмғқн•ңлӢӨ. мқҙлҘј the deadly triadлқјкі  л¶ҖлҘёлӢӨ.

  1. Function approximation
  2. Bootstrapping
  3. Off-policy learning

the deadly triad мӨ‘ 2к°Җм§Җ мҡ”мҶҢл§Ң кІ°н•©н•ңлӢӨл©ҙ л¶Ҳм•Ҳм •м„ұ мқҙмҠҲлҘј н”јн•  мҲҳ мһҲлӢӨ. л”°лқјм„ң мң„ 3к°ң мӨ‘ н•ҳлӮҳлҘј нҸ¬кё°н•  мҲҳ мһҲлӢӨл©ҙ нҸ¬кё°н•ҳлҠ” кІғмқҙ л°”лһҢм§Ғн•ҳлӢӨ.

лЁјм Җ, function approximationмқҖ лӢ№м—°нһҲ нҸ¬кё°н•  мҲҳ м—ҶлӢӨ. function approximationмқҖ л§Өмҡ° нҒ° state spaceлҘј нҡЁкіјм ҒмңјлЎң мІҳлҰ¬н•  мҲҳ мһҲлҠ” к°•л Ҙн•ң лҸ„кө¬мқҙкё° л•Ңл¬ёмқҙлӢӨ.

bootstrapping м—Ҷмқҙ н•ҷмҠөмқҙ к°ҖлҠҘн•ҳлӢӨлҠ” кІғмқҖ мқҙлҜё м•Ңл Өм§„ мӮ¬мӢӨмқҙлӢӨ. к°ҖмһҘ лҢҖн‘ңм Ғмқё кІҢ Monte Carlo (MC) MethodмқҙлӢӨ. к·ёлҹ¬лӮҳ bootstrapping methodлҠ” MC methodм—җ л№„н•ҙ нӣЁм”¬ к°•л Ҙн•ҳлӢӨ. onlineмңјлЎң н•ҷмҠөн•  мҲҳ мһҲкё° л•Ңл¬ём—җ нӣЁм”¬ л№ лҘҙкі , лҚ°мқҙн„°лҘј нҡЁмңЁм ҒмңјлЎң мӮ¬мҡ©н•ҳл©°, continuing taskм—җ м Ғмҡ© к°ҖлҠҘн•ҳлӢӨ. лҳҗн•ң MC methodлҠ” мқјл°ҳм Ғмқё bootstrapping methodм—җ л№„н•ҙ л¶„мӮ°мқҙ нӣЁм”¬ нҒ¬лӢӨ. л”°лқјм„ң bootstrapping м—ӯмӢң нҸ¬кё°н•ҳкё°лҠ” м–ҙл өлӢӨ.

л§Ҳм§Җл§үмңјлЎң off-policy learningмқҙлӢӨ. нҸ¬кё°н•  мҲҳ мһҲмқ„к№Ң? off-policy methodлҠ” target policyмҷҖ behavior policyк°Җ лӢӨлҘҙлӢӨ. мқҙлҠ” м–ҙл””к№Ңм§ҖлӮҳ нҺёлҰ¬н•ң кұ°м§Җ, л°ҳл“ңмӢң н•„мҡ”н•ң кІғмқҖ м•„лӢҲлӢӨ. к·ёлҹ¬лӮҳ лҚ” нҒ° лӘ©н‘ңлҘј к°Җм§„ к°•л Ҙн•ң AIлҘј л§Ңл“Өкё° мң„н•ҙм„ңлҠ” off-policy learningмқҖ н•„мҲҳм ҒмқҙлӢӨ. agentк°Җ н•ҳлӮҳмқҳ behavior policyлЎңл¶Җн„° м„ нғқлҗң actionмқ„ л°”нғ•мңјлЎң мғқм„ұлҗң experienceлҘј нҶөн•ҙ м—¬лҹ¬ к°ңмқҳ target policyлҘј лі‘л ¬м ҒмңјлЎң н•ҷмҠөн•  мҲҳ мһҲкё° л•Ңл¬ёмқҙлӢӨ.

Linear Value-function Geometry

мқҙ м„№м…ҳм—җм„ңлҠ” value function approximationмқ„ мЎ°кёҲ лҚ” м¶”мғҒм ҒмңјлЎң мқҙн•ҙн•ҙ ліј кІғмқҙлӢӨ. лЁјм Җ, state-value function $v : \mathcal{S} \rightarrow \mathbb{R}$мқҙ мһҲлӢӨкі  н•ҳмһҗ. лҳҗн•ң function approximationмқ„ мӮ¬мҡ©н•ҳлҠ” лҢҖл¶Җл¶„мқҳ кІҪмҡ°, state к°ңмҲҳліҙлӢӨ weight vector $\mathbf{w}$мқҳ weight к°ңмҲҳк°Җ нӣЁм”¬ мһ‘лӢӨ.

мҳҲлҘј л“Өм–ҙ 3к°ңмқҳ state $\mathcal{S} = \{ s_1, s_2, s_3 \}$мҷҖ 2к°ңмқҳ weight $\mathbf{w} = (w_1, w_2)^\top$к°Җ мһҲлӢӨкі  н•ҳмһҗ. мқҙ кІҪмҡ° value function $v = [v(s_1), v(s_2), v(s_3)]^\top$лҘј 3м°Ёмӣҗ кіөк°„ лӮҙмқҳ м җл“ӨлЎң н‘ңнҳ„лҗҳл©°, weight vectorлҠ” н•ҳмң„ кіөк°„мқё 2м°Ёмӣҗ кіөк°„мңјлЎң н‘ңнҳ„лҗңлӢӨ. м•„лһҳлҠ” мқҙм—җ лҢҖн•ң к·ёлҰјмқҙлӢӨ.

Fig 1. The geometry of linear value-function approximation.
(Image source: Sec 11.4 Sutton & Barto (2020).)

мң„ к·ёлҰјм—җм„ң true value function $v_\pi$лҠ” weight vector $\mathbf{w}$м—җ мқҳн•ҙ н‘ңнҳ„лҗҳлҠ” 2м°Ёмӣҗ кіөк°„ліҙлӢӨ лҚ” нҒ° кіөк°„м—җ (м—¬кё°м„ңлҠ” 3м°Ёмӣҗ) н‘ңнҳ„лҗңлӢӨ.

м–ҙл–Ө кі м •лҗң policy $\pi$лҘј кі л Өн•ҙліҙмһҗ. true value function $v_\pi$лҠ” мң„ к·ёлҰјмІҳлҹј л§Өмҡ° ліөмһЎн•ҙм„ң weight vector $\mathbf{w}$лЎң м •нҷ•нһҲ к·јмӮ¬н•ҳлҠ” кІғмқҖ л¶Ҳк°ҖлҠҘн•ҳлӢӨ. мҰү, $v_\pi$лҠ” н•ҳмң„ кіөк°„м—җ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ”лӢӨ. к·ёл ҮлӢӨл©ҙ н‘ңнҳ„ к°ҖлҠҘн•ң value function мӨ‘ м–ҙл–Ө кІғмқҙ true value functionм—җ к°Җк№Ңмҡёк№Ң?

лЁјм Җ, л‘җ value function мӮ¬мқҙмқҳ кұ°лҰ¬лҘј мёЎм •н•  н•„мҡ”к°Җ мһҲлӢӨ. л‘җ value function $v_1$, $v_2$к°Җ мЈјм–ҙмЎҢмЎҢмңјл©° $v = v_1 - v_2$лқјкі  н•ҳмһҗ. лҳҗн•ң нҠ№м • stateлҘј мЎ°кёҲ лҚ” м •нҷ•нһҲ м¶”м •н•ҳкё° мң„н•ҙ к·ё stateм—җ м–јл§ҲлӮҳ м§‘мӨ‘н•  м§ҖлҘј лӮҳнғҖлӮҙлҠ” state distribution $\mu : \mathcal{S} \rightarrow [0,1]$лҘј (мЈјлЎң on-policy distributionмқҙ мӮ¬мҡ©лҗЁ) мӮ¬мҡ©н•ҳмһҗ. мқҙ л•Ң value function мӮ¬мқҙмқҳ distanceлҘј м•„лһҳмҷҖ к°ҷмқҙ normмқ„ мӮ¬мҡ©н•ҙ м •мқҳн•  мҲҳ мһҲлӢӨ.

\[\big\lVert v \big\rVert_\mu^2 \doteq \sum_{s \in \mathcal{S}} \mu(s) v(s)^2\]

мң„ м •мқҳлҘј мӮ¬мҡ©н•  л•Ң mean square value error $\overline{\text{VE}}(\mathbf{w}) \doteq \sum_{s \in \mathcal{S}}\mu(s)\Big[v_\pi(s) - \hat{v}(s,\mathbf{w}) \Big]^2$лҘј $\overline{\text{VE}}(\mathbf{w}) = \lVert v_{\mathbf{w}} - v_\pi \rVert_\mu^2$мҷҖ к°ҷмқҖ лӢЁмҲңн•ң нҳ•нғңлЎң лӮҳнғҖлӮј мҲҳ мһҲлӢӨ.5 м–ҙл–Ө value functionмқҳ н•ҳмң„ кіөк°„м—җм„ңмқҳ к°ҖмһҘ к°Җк№Ңмҡҙ н‘ңнҳ„ к°ҖлҠҘн•ң value functionмқҖ projection м—°мӮ° $\Pi$мқ„ нҶөн•ҙ м°ҫмқ„ мҲҳ мһҲлӢӨ.

\[\Pi v \doteq v_\mathbf{w} \text{ where } \mathbf{w} = \underset{\mathbf{w} \in \mathbb{R}^d}{\arg\min} \ \big\lVert v - v_\mathbf{w} \big\rVert_\mu^2\]

true value function $v_\pi$м—җ к°ҖмһҘ к°Җк№Ңмҡҙ н‘ңнҳ„ к°ҖлҠҘн•ң value functionмқҖ $v_\pi$мқҳ projection $\Pi v_\pi$лЎң Fig 1м—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

м—¬кё°м„ңлҠ” мқҙм •лҸ„ м»Ём…үл§Ң мқҙн•ҙн•ҳкі  л„ҳм–ҙк°„лӢӨ. лҚ” мһҗм„ён•ң лӮҙмҡ©мқҖ м—¬кё°м„ң мҶҢк°ңн•ҳкё°м—җлҠ” л„Ҳл¬ҙ ліөмһЎн•ҙ к¶ҒкёҲн•ҳлӢӨл©ҙ Sutton мұ…мқ„ м°ёмЎ°н•ҳкё° л°”лһҖлӢӨ.6
